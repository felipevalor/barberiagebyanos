# Flujo de WhatsApp — Del primer mensaje al cierre

> **Estructura:** Calificar → Demo → Precio → Cierre → Seguimiento
> Cada etapa tiene un objetivo claro. No saltear pasos.

---

## ETAPA 1 — Primer mensaje en WhatsApp

Mandás vos cuando te pasan el número (o cuando te escriben ellos):

```
Hola [Nombre]! Soy Felipe de Valor Solutions 👋

Te mando el link del sitio de Gebyanos para que lo pruebes
como si fueras un cliente — entrá desde el celu:

👉 https://barberia-d8q.pages.dev/

Reservá un turno de prueba y fijate qué pasa.
```

> Silencio. Dejá que lo prueben. No expliques nada todavía.

---

## ETAPA 2 — Calificación (después de que ven la demo)

```
¿Cómo te pareció?

Y contame — ¿cómo manejás los turnos hoy?
```

**Escuchá la respuesta y respondé según el caso:**

---

### Caso A: "Por DM / WhatsApp / llamada"
→ Dolor máximo. Cerrás más rápido.

```
Clásico. El problema es que dependés de estar disponible para responder —
y los clientes no tienen forma de saber si hay lugar antes de escribirte.

Con esto el cliente ve los horarios disponibles en tiempo real,
elige, y el turno te llega confirmado al WhatsApp. Vos no tenés que hacer nada.
```

### Caso B: "Usamos Reservo / Fresha / TurnoWeb"
→ Diferenciarte de las apps genéricas.

```
Entiendo. El problema con esas plataformas es que el cliente tiene que
registrarse, crear cuenta, y todo queda bajo la marca de la app — no la tuya.

Acá el cliente entra a TU sitio, con TU identidad. Sin app, sin registro.
Y el turno te llega directo a WhatsApp, que ya usás para todo.
```

### Caso C: "No tomamos turnos, vienen directo"
→ Plantear el costo invisible.

```
Tiene sentido si el flujo funciona. Pero hay clientes que no van porque
no saben si hay lugar — especialmente a la tarde o los sábados.

Con turnos online captás esa demanda que hoy se pierde porque
el cliente no se anima a ir sin saber.
```

## ETAPA 3 — El pitch después de calificar

```
Lo que hacemos en Valor Solutions es esto:

✅ Sitio web premium hecho a medida para tu barbería
✅ Sistema de turnos que se sincroniza con tu calendario
✅ El cliente reserva en 30 segundos — sin app, sin registro
✅ El turno te llega a WhatsApp con nombre, servicio y horario
✅ Panel de admin para ver todos los turnos del día

Todo configurado. Vos solo tenés que empezar a recibir clientes.
```

---

## ETAPA 4 — El precio

No lo escondas. Sé directo:

```
El costo es:

🔹 Setup inicial (diseño + configuración): $[PRECIO]
🔹 Mantenimiento mensual (hosting + soporte): $[PRECIO]/mes

El primer mes sin cargo para que lo pruebes con tus clientes reales.

¿Querés que te arme un presupuesto formal con todo el detalle?
```

> **Tip:** Si preguntaron cuánto sale antes de que termines el pitch, respondé:
> "Depende de lo que necesitás — primero contame cómo funciona el negocio y
> te doy un número exacto."

## ETAPA 5 — Cierre

```
¿Arrancamos? Para empezar necesito:
- Nombre de la barbería y logo
- Foto del local o del equipo
- Horarios de atención
- Número de WhatsApp de cada barbero

En una semana tenés el sitio funcionando.
```

## ETAPA 6 — Seguimiento si se enfrían (3-5 días sin respuesta)

**Día 3:**
```
[Nombre], quedé a la espera — ¿tuviste chance de verlo?
```

**Día 5:**
```
Te mando esto por si sirve: el sitio de Gebyanos tiene
[X] reservas en el último mes que antes llegaban por DM o no llegaban.

Si en algún momento querés charlarlo, acá estoy.
```

**Día 10 (último intento):**
```
[Nombre], último mensaje de mi parte para no molestarte.

Si el timing no es el correcto ahora, guardá el contacto —
cuando quieras armarlo lo hacemos en una semana.

Éxitos con la barbería 🤝
```

> No te disculpes, no presiones, no hagas descuento desesperado.
> La escasez funciona mejor que el descuento.
