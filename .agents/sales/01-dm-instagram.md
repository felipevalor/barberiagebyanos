# DM de Instagram — Primer contacto

> **Regla de oro:** El objetivo del DM no es vender. Es conseguir que pasen a WhatsApp.
> Un solo mensaje. Sin adjuntos. Sin pitch largo. Sin precio.

---

## Mensaje A — Para barberías que dicen "turnos por DM" en la bio

```
Hola [Nombre], vi la barbería — muy buena onda.

Hice el sitio de Gebyanos acá en Rosario. Los clientes reservan turno directo
desde el celu, sin app ni registro, y el turno llega al WhatsApp del barbero
confirmado. Ya no responden DMs para turnos.

¿Te muestro cómo quedó?
```

---

## Mensaje B — Para barberías sin sistema visible de turnos

```
Hola [Nombre], caí en tu perfil y me gustó la vibra del local.

Hago sitios web para barberías con sistema de turnos por WhatsApp integrado —
sin app, sin registro para el cliente. Lo hice para Gebyanos en Rosario y les
cambió bastante la operación.

¿Te mando el link para que lo veas?
```

---

## Mensaje C — Para barberías con muchos seguidores pero sin web

```
Hola [Nombre] — con el nivel de contenido que tenés merece un sitio web a la altura.

Hago landing pages premium para barberías con turno online integrado.
El caso más reciente es Gebyanos (Rosario): los clientes reservan en 30 segundos
y el turno llega directo al WhatsApp del barbero.

¿Te paso el link para que lo pruebes como si fueras un cliente?
```

```
[Nombre], te debo el link del sitio que te comenté.
Si querés lo ves cuando tengas 2 minutos: [link demo]
```

